# 注册功能测试

## 账号注册

### 用户名输入
#### 正常场景
- 输入合法长度的用户名（如 4-16 位）
- 输入纯字母用户名
- 输入字母+数字组合用户名
- 输入含下划线等允许字符的用户名
#### 异常场景
- 用户名为空
- 输入已注册的用户名（重复性校验）
- 用户名长度不足最低限制
- 用户名超过最大长度限制
- 输入纯数字用户名（若不支持）
- 输入含特殊字符（如 @、#、空格）
- 输入 SQL 注入语句
- 输入 XSS 脚本

### 登录密码输入
#### 正常场景
- 输入符合规则的密码（如 8-16 位含字母数字）
- 点击眼睛图标切换明文/密文显示
#### 异常场景
- 密码为空
- 密码长度不足最低要求
- 密码超过最大长度限制
- 密码不符合复杂度要求（纯数字/纯字母）
- 密码含特殊字符是否支持
- 输入超长字符串

### 图形验证码
#### 正常场景
- 输入正确图形验证码（如 4425）
- 验证码不区分大小写
#### 异常场景
- 验证码为空
- 输入错误验证码，给出提示
- 验证码过期后提交
- 刷新验证码后旧码是否失效
- 验证码刷新功能是否正常

### 服务协议与隐私政策
#### 正常场景
- 勾选复选框后方可注册
- 点击《服务协议》跳转对应页面
- 点击《隐私政策》跳转对应页面
#### 异常场景
- 未勾选复选框直接点击注册，给出提示
- 复选框默认状态是否为未勾选

### 注册按钮
#### 正常场景
- 所有字段合法且勾选协议，点击注册成功
- 注册成功后跳转逻辑正确（如自动登录或跳转登录页）
#### 异常场景
- 任意字段为空时点击注册给出对应提示
- 多次快速点击注册按钮（防重复提交）
- 网络异常时点击注册的错误处理

## 手机注册

### 手机号输入
#### 正常场景
- 输入已注册的 11 位大陆手机号格式
- 输入未注册的合法手机号
#### 异常场景
- 手机号为空
- 输入已注册手机号（重复性校验）
- 输入非 11 位手机号
- 输入含字母或特殊字符
- 输入港澳台/海外号码格式
- 输入 SQL 注入/XSS 脚本

### 验证码获取
#### 正常场景
- 手机号合法时点击"获取验证码"，短信正常下发
- 倒计时 60s 内按钮置灰不可重复点击
- 倒计时结束后按钮恢复，可重新获取
#### 异常场景
- 手机号为空时点击获取验证码给出提示
- 手机号不合法时点击获取验证码给出提示
- 频繁请求是否有频率限制（如 1 分钟内仅能发 1 次）
- 同一手机号短时间多次请求是否触发风控
- 验证码有效期是否为 5 分钟

### 验证码输入
#### 正常场景
- 输入正确验证码通过校验
#### 异常场景
- 验证码为空时点击注册

- 输入已过期的验证码
- 验证码重复使用是否失效

## 兼容性

### 浏览器兼容
- Chrome / Firefox / Edge 正常显示与操作
- 移动端 H5 适配，布局无错位

### 分辨率适配
- 不同屏幕尺寸下页面布局正常

## 安全性

### 数据传输
- 注册接口是否走 HTTPS
- 密码是否明文传输（抓包验证）

### 输入安全
- 所有输入框防 SQL 注入
- 所有输入框防 XSS 攻击

### 验证码安全
- 验证码绑定账号（A 的验证码不可用于 B）
- 验证码一次性，使用后立即失效
- 验证码有时效限制

### 频率限制
- 同一 IP 频繁注册是否触发限制
- 同一手机号/邮箱短时间内多次请求验证码是否触发风控

## 性能

### 响应速度
- 注册接口响应时间 < 2s
- 获取验证码接口响应时间 < 3s

### 并发
- 高并发注册场景下服务稳定
- 同一账号并发注册不产生重复数据
