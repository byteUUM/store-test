# 登录功能测试

## 账号密码登录

### 账号输入
#### 正常场景
- 输入已注册的有效账号
- 账号为手机号格式
- 账号为邮箱格式
- 账号为用户名格式
#### 异常场景
- 账号为空
- 输入未注册的账号
- 输入含特殊字符的账号
- 输入超长字符串
- 输入 SQL 注入语句
- 输入 XSS 脚本

### 密码输入
#### 正常场景
- 输入正确密码
- 点击眼睛图标切换明文/密文显示
#### 异常场景
- 密码为空
- 输入错误密码
- 密码错误达到限制次数（是否锁定账号）
- 输入超长密码
- 密码含特殊字符

### 图形验证码
#### 正常场景
- 输入正确验证码
- 验证码不区分大小写
#### 异常场景
- 验证码为空
- 输入错误验证码
- 验证码过期后提交
- 刷新验证码后旧码是否失效

### 登录按钮
#### 正常场景
- 三项全部填写正确，点击登录成功跳转
#### 异常场景
- 任意字段为空时点击登录
- 多次快速点击登录按钮（防重复提交）

## 手机验证码登录

### 手机号输入
#### 正常场景
- 输入已注册的有效手机号
- 输入 11 位标准大陆手机号
#### 异常场景
- 手机号为空
- 输入未注册手机号
- 输入非 11 位手机号
- 输入字母或特殊字符
- 输入港澳台/海外号码格式

### 验证码获取
#### 正常场景
- 手机号合法时点击"获取验证码"，短信正常下发
- 倒计时 60s 内按钮不可重复点击（置灰）
- 倒计时结束后可重新获取
#### 异常场景
- 手机号为空时点击获取验证码
- 手机号不合法时点击获取验证码
- 频繁请求验证码（是否有频率限制）
- 验证码短信是否在有效期内（通常 5 分钟）

### 验证码输入
#### 正常场景
- 输入正确验证码，点击登录成功
#### 异常场景
- 验证码为空
- 输入错误验证码
- 输入过期验证码
- 验证码重复使用（用过的码是否失效）

## 邮箱验证码登录

### 邮箱输入

- 倒计时内按钮置灰不可重复点击
- 倒计时结束后可重新获取
#### 异常场景
- 邮箱为空时点击获取
- 邮箱不合法时点击获取
- 频繁请求是否有限制
- 邮件是否落入垃圾箱（提示文案）

### 验证码输入
#### 正常场景
- 输入正确验证码，点击登录成功
#### 异常场景
- 验证码为空
- 输入错误验证码
- 输入过期验证码
- 验证码重复使用是否失效

## Tab 切换

### 正常场景
- 三个 Tab 可正常切换
- 切换后下划线指示正确高亮
- 切换 Tab 后已输入内容是否清空

### 异常场景
- 快速连续切换 Tab 不出现错乱
- 切换 Tab 后获取验证码的倒计时是否重置

## 找回密码（重置密码）

### 入口
- 登录页点击"找回密码"可正常跳转重置密码页面
- 页面标题显示"重置密码"

### 手机/邮箱输入
#### 正常场景
- 输入已注册的手机号，格式合法
- 输入已注册的邮箱，格式合法
- 同时支持手机号和邮箱两种格式识别
#### 异常场景
- 输入框为空时点击获取验证码
- 输入未注册的手机号/邮箱
- 输入格式不合法的手机号（非 11 位、含字母）
- 输入格式不合法的邮箱（缺少@或域名）
- 输入超长字符串
- 输入 SQL 注入/XSS 脚本

### 验证码获取
#### 正常场景
- 手机号/邮箱合法时点击"获取验证码"成功下发
- 手机号场景：短信正常送达
- 邮箱场景：邮件正常送达，未落入垃圾箱
- 倒计时期间按钮置灰，不可重复点击
- 倒计时结束后按钮恢复，可重新获取
#### 异常场景
- 字段为空时点击获取验证码给出提示
- 频繁点击是否有频率限制（如 1 分钟内只能发 1 次）
- 同一账号短时间内多次请求是否触发风控
- 验证码有效期是否为 5 分钟（过期后不可使用）

### 验证码输入
#### 正常场景
- 输入正确验证码通过校验
#### 异常场景
- 验证码为空时点击确认
- 输入错误验证码给出明确提示
- 输入已过期的验证码
- 输入已使用过的验证码（不可二次使用）
- 连续输错多次是否触发锁定或重新发码限制

### 新密码输入
#### 正常场景
- 输入符合规则的新密码（如 8-16 位含字母数字）
- 点击眼睛图标切换密码明文/密文显示
- 新密码与原密码不同时允许提交
#### 异常场景
- 密码为空时点击确认
- 密码长度不足最低要求
- 密码不符合复杂度要求（纯数字/纯字母等）
- 密码超过最大长度限制
- 新密码与原密码相同（是否有限制）
- 输入含特殊字符是否支持

### 确认按钮
#### 正常场景
- 三项全部正确填写，点击确认重置成功
- 重置成功后提示语清晰（如"密码修改成功"）
- 重置成功后自动跳转到登录页或自动登录
#### 异常场景
- 任意字段为空时点击确认给出对应提示
- 多次快速点击确认按钮（防重复提交）
- 网络异常时点击确认的错误处理

### 重置后验证
- 使用旧密码登录失败（旧密码已失效）
- 使用新密码可正常登录
- 其他已登录设备是否被强制下线（可选）

### 页面跳转
- 点击"立即登录"跳转至登录页面
- 返回按钮/浏览器回退行为是否正常

### 安全性
- 重置密码接口是否走 HTTPS
- 新密码是否明文传输（抓包验证）
- 验证码是否绑定账号（A 的验证码不能用于 B）
- 重置链接/验证码是否一次性且有时效

## 其他功能

### 找回密码入口
- 点击"找回密码"跳转正确页面

### 立即注册
- 点击"立即注册"跳转注册页面

### 兼容性
- Chrome / Firefox / Edge 浏览器
- 移动端 H5 适配
- 不同分辨率下布局不错位

### 安全性
- 登录接口是否走 HTTPS
- 密码是否明文传输（抓包验证）
- 连续失败是否触发账号锁定或人机验证
- Cookie / Token 是否设置合理过期时间

### 性能
- 登录接口响应时间 < 2s
- 高并发下登录是否正常
